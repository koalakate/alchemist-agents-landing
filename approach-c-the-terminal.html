YOU
Convert this SAS program to PySpark. It filters high-horsepower cars and adds pricing columns.
SAS — input.sas
data work.high_power_cars;
  set sashelp.cars;
  where horsepower > 300;
  tax_category = 'Premium';
  adjusted_msrp = msrp * 1.05;
run;
● agent delegated to alchemist mcp — converting...
AI
Done. Here's your converted PySpark code. The SAS data step has been translated to DataFrame operations with filter, withColumn, and lit.
PySpark — output.py   ✓ verified
# Converted by Alchemist
from pyspark.sql import functions as F

df = spark.table("sashelp.cars")
df = df.filter(F.col("horsepower") > 300)
df = df.withColumn("tax_category", F.lit("Premium"))
df = df.withColumn("adjusted_msrp", F.col("msrp") * 1.05)
